# Advanced Cab Booking System (C++)
This is a console-based cab booking system built using Object-Oriented Programming concepts.

# Features
- Book a cab ride
- Driver allocation system
- Cab type selection (Mini, Sedan, SUV)
- Fare calculation based on distance
- Ride history using file handling

# Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

# How to Run
1. Compile the code using g++
2. Run the executable file
